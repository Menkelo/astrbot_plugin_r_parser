# [v1.2.0] - B站 / 抖音解析能力强化与结构优化

* **🚀 Bilibili 解析提速**
  * 新增视频信息与播放地址轻量缓存（`get_info / get_download_url`）
  * 减少重复请求带来的等待时间，重复解析同链接响应更快
  * 评论卡片渲染改为异步任务，避免阻塞主视频主链路

* **🧹 Bilibili 评论区质量过滤（双层）**
  * 第一层：文本规则过滤（引流词、联系方式、短链等）
  * 第二层：二维码图片识别过滤（可选依赖 OpenCV + numpy）
  * 支持二维码检测次数上限与超时控制，兼顾效果与性能

* **🧩 Bilibili 结构分层**
  * 拆分评论相关能力至独立模块：
    * `comment_service.py`
    * `comment_renderer.py`
    * `stream_selector.py`
  * 主解析器 `__init__.py` 聚焦路由与流程编排，可维护性提升

* **🎬 Douyin 多段动态视频解析增强**
  * 支持短链场景下的多段动态图（`images[].video`）提取
  * 修复仅识别单条媒体的问题
  * 动态视频去重逻辑增强（`uri / video_id / fallback key`）

* **📦 Douyin 下载稳定性修复**
  * 修复动态视频文件同名覆盖（如 `play.mp4`）导致丢失/重复的问题
  * 为动态视频分段下载引入唯一命名策略
  * 图集含动态视频时可按策略优先发送视频内容

* **🎞️ Douyin 合并能力（可选）**
  * 支持多段动态图自动合并为单视频
  * 支持提取背景音乐并尝试合成到合并后视频
  * 兼容无音乐场景自动回退无音轨合并结果

* **🛠️ 下载兜底与健壮性**
  * 下载链路保持多级兜底：`curl_cffi -> aiohttp -> yt-dlp`
  * 强化异常处理与日志可观测性，便于排障
